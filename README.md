## 解决了哪些问题？

1. 全局导航
2. 页面管理
3. 转场效果
4. 触摸手势
5. 页面滚动

## 有哪些主要的特性？

1. 只需用业务代码，即可轻松搭建WebApp。
2. 灵活的插件机制，易于二次开发，扩展业务功能。
3. 对第三方框架友好，不强奸开发者喜好。
4. 为各平台提供统一方案，保证兼容性。
